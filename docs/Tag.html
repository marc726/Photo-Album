Tag

The Tag class represents a tag with attributes. It implements Serializable.

Fields

  • tagName: String
  • tagValue: String

Methods

Tag(String tagName, String tagValue)

Constructor that initializes the tag with the given name and value.

getTagName()

Returns the name of the tag.

getTagValue()

Returns the value of the tag.

toString()

Returns a string representation of the tag, in the format "name=value".

equals(Object obj)

Checks if this tag is equal to another object. Returns true if the object is a tag with the same name and value, and false otherwise.

hashCode()

Returns a hash code value for the tag, which is based on the name and value.
