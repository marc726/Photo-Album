Photo

The Photo class represents a photo with attributes. It implements Serializable.

Fields

  • tags: ArrayList<Tag>
  • name: String
  • caption: String
  • date: LocalDateTime
  • imagePath: String

Methods

Photo(String name, LocalDateTime date)

Constructor that initializes the photo with the given name and date, an empty caption, and an empty list of tags.

Photo()

Default constructor that initializes the photo with an empty name and caption, the current date and time, and an empty list of tags.

getName()

Returns the name of the photo.

setName(String name)

Sets the name of the photo.

getCaption()

Returns the caption of the photo.

setCaption(String caption)

Sets the caption of the photo.

getDate()

Returns the date of the photo.

setDate(LocalDateTime date)

Sets the date of the photo.

getTags()

Returns the list of tags of the photo.

getImagePath()

Returns the image path of the photo.

setImagePath(String imagePath)

Sets the image path of the photo.

addTag(Tag tag)

Adds a tag to the photo if it does not already exist.
