PhotoController

The PhotoController class is responsible for controlling the display of a single photo.

Fields

  • photoImageView: ImageView
  • captionLabel: Label
  • dateTimeLabel: Label
  • tagsLabel: Label

Methods

setPhoto(Photo photo)

Sets the photo to be displayed in the controller. The photo object to be displayed is passed as a parameter. The method sets the photo image, caption, date-time of capture, and tags.
