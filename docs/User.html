User

The User class represents a user with a username. It implements Serializable.

Fields

  • username: String
  • albums: ArrayList<Album>

Methods

User(String username)

Constructor that initializes the user with the given username and an empty list of albums.

getUsername()

Returns the username of the user.

setUsername(String username)

Sets the username of the user.

toString()

Returns a string representation of the user, which is the username.

getAlbums()

Returns the list of albums of the user.
