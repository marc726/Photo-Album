Album

The Album class represents an album with attributes. It implements Serializable.

Fields

  • albumName: String
  • photos: List<Photo>

Methods

Album(String albumName)

Constructor that initializes the album with the given name and an empty list of photos.

getAlbumName()

Returns the name of the album.

setAlbumName(String name)

Sets the name of the album.

getPhotos()

Returns the list of photos in the album.

getNumPhotos()

Returns the number of photos in the album.

addPhoto(Photo photo)

Adds a photo to the album.

removePhoto(Photo photo)

Removes a photo from the album if it exists.

toString()

Returns a string representation of the album, including the name, photo count, date of the oldest photo, and date of the newest photo.
