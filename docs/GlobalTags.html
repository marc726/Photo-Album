GlobalTags

The GlobalTags class represents a singleton instance that stores a set of tag types. It provides methods to access and modify the tag types.

Fields

  • instance: GlobalTags
  • tagTypes: Set<String>

Methods

getInstance()

Returns the singleton instance of GlobalTags. If the instance does not exist, it is created.

getTagTypes()

Returns the set of tag types.

setTagTypes(Set<String> tagTypes)

Sets the set of tag types.

addTagType(String tagType)

Adds a new tag type to the set of tag types.
